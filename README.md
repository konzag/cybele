     /\   /\

   /`       `\
  | ΚΥΒΕΛΙΤΣΑ |
   \_________/

---

## Τι είναι αυτό;

Δύο παιχνίδια φτιαγμένα ειδικά για την **Κυβελίτσα** (3 ετών) —
τη μικρή μας κυνηγό δαιμόνων! 🎉

Θέμα: **Κυνήγι Δαιμόνων** — χαριτωμένο, πολύχρωμο, ποτέ τρομακτικό.
Επίπεδο: **Νηπιαγωγείο** (3 ετών).

---

## 📁 Αρχεία

| Αρχείο | Τι κάνει |
|--------|----------|
| `index.html` | Παιχνίδι στον browser (κινητό / tablet / υπολογιστής) |
| `cybele.py`  | Παιχνίδι στο terminal των Windows |

---

## 🌐 index.html — Παιχνίδι στο Browser

### Πώς να το τρέξεις

Απλώς **κάνε διπλό κλικ** στο αρχείο `index.html`
ή σύρε το μέσα σε Chrome / Edge / Firefox.

**Δεν χρειάζεται internet!** Λειτουργεί εντελώς offline.

### Τι περιέχει

- **👾 Κυνήγι Δαιμόνων** — Χαριτωμένα δαιμονάκια εμφανίζονται στην οθόνη.
  Πάτα τα να τα πιάσεις! Μετράει πόσα έπιασες.

- **🎨 Χρώματα & Αριθμοί** — Κάθε δαιμονάκι έχει χρώμα και αριθμό (1-5).
  Πριν εξαφανιστεί, λέει το χρώμα και τον αριθμό του στα ελληνικά.

- **🌙 Ημέρα / Νύχτα** — Κουμπί για εναλλαγή. Τη νύχτα τα δαιμονάκια
  λάμπουν και ακολουθεί φακός το δάχτυλό σου!

- **🎉 Γιορτή** — Όταν πιάσεις όλα: κομφετί, αστέρια και μεγάλο μπράβο!

### Συμβατότητα

Λειτουργεί σε κινητό, tablet και υπολογιστή.
Μεγάλα κουμπιά για μικρά δαχτυλάκια! 🤏

---

## 🐍 cybele.py — Παιχνίδι στο Terminal

### Απαιτήσεις

- Python 3.7+
- Windows (για τους ήχους με `winsound`)
- `colorama` (προαιρετικό, για χρώματα):

```bash
pip install colorama
```

### Πώς να το τρέξεις

```bash
python cybele.py
```

Ή διπλό κλικ αν έχεις Python συνδεδεμένο με `.py` αρχεία.

### Μενού

| Επιλογή | Παιχνίδι |
|---------|----------|
| `1` | 👾 Κυνήγι Δαιμόνων — Μετράμε δαιμονάκια μαζί (1–5) |
| `2` | 🎨 Χρώματα — Έγχρωμα μπλοκ και ερωτήσεις |
| `3` | 🔢 Αριθμοί — ASCII αριθμοί 1-5 με αστεράκια |
| `4` | 🎵 Τραγουδάκι — Μελωδία με `winsound.Beep()` |
| `5` | 👋 Αποχαιρετισμός — Γεια σου Κυβελίτσα! |
